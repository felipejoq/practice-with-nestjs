## Description
Practicando con Nest.js para construir una simple REST ful API

## Installation

```bash
$ yarn install
```
## Populate Data from Seed:
```bash
Get -> http://localhost:3000/seed
```
### Get with curl:
```bash
curl http://localhost:3000/seed
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
